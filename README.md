# Anemia Analysis Project
Analysis of factors influencing anemia using statistical and machine learning techniques.

## 📊 Overview

This project analyzes the factors influencing the prevalence of anemia in women and children using data from the National Family Health Survey (NFHS). It combines statistical analysis and machine learning techniques to identify key determinants of anemia.

## 🚀 Project Highlights
- Identified key socio-economic determinants of anemia
- Applied Chi-Square test for statistical significance
- Built and compared multiple machine learning models
- Achieved best performance using Random Forest

## 🎯 Objectives

* Analyze prevalence and severity of anemia
* Study relationship with socio-economic factors (education, wealth index, residence)
* Perform statistical analysis using Chi-Square test
* Build machine learning models to predict anemia levels

## 📁 Dataset

The dataset is obtained from NFHS and includes:

* Education level
* Wealth index
* Place of residence
* Health and nutritional indicators

## 🔍 Methodology

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Statistical Analysis (Chi-Square Test)
* Machine Learning Models:

  * Logistic Regression
  * Decision Tree
  * Random Forest

## 📈 Key Findings

* Lower education and wealth levels are associated with higher anemia prevalence
* Rural areas show slightly higher anemia levels
* Random Forest performed best among all models

## 🛠️ Tools Used

- R Programming (Data Analysis & Visualization)
- Statistical Analysis (Chi-Square Test)
- Machine Learning (Logistic Regression, Decision Tree, Random Forest)
